Touch vs Click

Left = try. Right = log.

1. Tap

Log prefix [1 tap]. Listeners use element.addEventListener(type, fn) — same pattern for "click" as for "touchstart", etc. (see demo.js). Same TAP_EVENT_TYPES as section 2: pointerdown/pointerup/pointercancel, touchstart/touchend/touchcancel, mousedown/mouseup (DOM mouse event types via addEventListener — not the HTML onclick attribute), click — many lines from one tap or click.

2. Drag (same listeners)

Prefix [2 drag]. The thumb uses the same listeners as section 1, plus pointermove for position (not logged). Tap the thumb once, then drag it — compare to [1 tap] (often click changes or disappears after a real drag).
